# Skill: backend-fastapi

## 适用范围

- FastAPI 路由开发
- SQLAlchemy 模型与服务层实现
- Alembic 迁移
- Celery 异步任务

## 规则

- 路由层只做参数接收、权限检查入口和响应拼装。
- 业务状态流转必须放在服务层。
- 审计事件写入必须统一走审计服务，不要散落在路由中。
- 任何新增表结构都必须同步 Alembic 迁移。
- 使用 `uv` 管理 Python 依赖和运行命令。

## 当前后端实现重点

- 文档上传与版本管理
- 流转状态机
- 毕业交接
- 负责人总览
- AI 建议存储与查询
- 群晖 / MinIO 存储适配层
